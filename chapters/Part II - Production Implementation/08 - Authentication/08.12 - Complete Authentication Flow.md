# Layer Responsibilities

| Layer      | Responsibility                 |
| ---------- | ------------------------------ |
| Route      | Maps URL to Controller         |
| Middleware | Authentication & Authorization |
| Controller | Handle Request & Response      |
| Service    | Business Logic                 |
| Repository | Database Communication         |
| Prisma     | ORM                            |
| PostgreSQL | Persistent Storage             |

Each layer should only perform its own responsibility.

# Production Security Checklist

Always

‚úì Hash Passwords using bcrypt

‚úì Use HTTPS

‚úì Store JWT Secrets in Environment Variables

‚úì Keep Access Tokens Short-Lived

‚úì Store Refresh Tokens Securely

‚úì Rotate Refresh Tokens

‚úì Revoke Tokens During Logout

‚úì Validate Every Request

‚úì Never Trust Client IDs

‚úì Implement Authorization

‚úì Log Authentication Failures

‚úì Rate Limit Login Endpoints

‚úì Validate Request Bodies

‚úì Sanitize Responses
